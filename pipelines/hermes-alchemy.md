## The 7 Hermetic Operations

Based on the 7 operations of the Emerald Tablet — but applied to content intelligence, not metal.

### 1. Calcination (Reddit Mining)

**What it is:** Burning away the dross to reveal the fixed essence.

Reddit is raw ore — 84 GB of noise, argument, self-promotion, and occasional signal. Calcination burns away the ephemeral (single appearances, low-effort posts, self-promotion) and leaves the fixed (recurring questions across years, cross-subreddit patterns, persistent confusion).

```
Prima Materia: 30 subreddits, 217 Parquet files
Flame:         DuckDB queries filtering by subreddit, score, year
Fixed Essence: question clusters with frequency × persistence × spread

Subroutines:  REDDIT_CLAIM_EXTRACTION (Proposals 2, 7, 1)
Gate:         ≥10 topics with cross_subreddit ≥ 3
```

### 2. Dissolution (Signal Separation)

**What it is:** Dissolving the fixed into its constituent elements so each can be examined separately.

Each candidate topic is dissolved across three solvents — YouTube gap data, YouNiverse breakout, Google Trends — to reveal its true constitution. A topic that survives all three solvents has genuine substance.

```
Solvent 1: YouTube gap scan    — is it underserved?
Solvent 2: YouNiverse breakout — has it worked elsewhere?
Solvent 3: Trends velocity     — is it entering mass consciousness?

What dissolves:  topics sustained by coincidence, API noise, single-subreddit bubbles
What remains:    topics with genuine demand × scarcity

Subroutines:  YOUTUBE_GAP_SCAN, GOOGLE_TRENDS_SCAN, YOUNIVERSE_CHECK
Gate:         ≥5 topics with gap + breakout + velocity
```

### 3. Separation (Topic → Treatment)

**What it is:** Drawing the subtle from the gross — separating the form from the matter.

The topic (matter) is separated from its packaging (form). The same topic can be framed as a question, a revelation, a warning, a comparison. The separation operation identifies which form fits which matter.

```
Matter:  "Why awakening feels like depression"
Forms tested by reusable subroutines:

  HEADLINE_ENGINE (Upworthy posteriors, reusable across ALL farms):
    - Generates 20 title variants
    - Scores by P(win) from causal A/B data
    - "actually" bonus (+0.48), question penalty (-0.27)
    - Returns top 3 with confidence

  CLICKSTREAM_GATEWAY (reusable):
    - "Chinnamasta" → gateway entity "Kali" (familiarity 0.82)
    - Gateway framing: "What Kali's Severed Head Tells Us About Chinnamasta"

  NARRATIVE_FRAME_LIBRARY (reusable):
    - misconception_reversal: "What If Everything You Know About Awakening Is Wrong?"
    - forbidden_knowledge: "Why Monks Don't Talk About This Stage of Awakening"
    - personal_discovery: "I Thought Enlightenment Would Fix Me. It Didn't."

Output: treatment_package (topic separated into its optimal form)
Gate:   treatment_package produced
```

### 4. Conjunction (Topic × Packaging × Source)

**What it is:** The sacred marriage — reuniting the separated elements in a new, higher form.

The topic, its packaging, and its scholarly source are married. This is where the celestial (RO library, source texts) meets the terrestrial (Reddit question, mass audience framing). The conjunction is the essay — the first product that bridges both worlds.

### 5. Fermentation (Script Generation)

**What it is:** The putrefaction and rebirth — the compost heap where old forms die and new life emerges.

The essay (dead text on a page) is buried in the fermentation vessel (the script pipeline). It decomposes into its elements — claims, quotes, objections, resolutions — and is reborn as something living: a script with two voices, rhythm, tension, and release.

The 8-beat format is the fermentation vessel:

```
BEAT 1: Hook (calcined from Reddit ore — the most attention-efficient form)
BEAT 2: Context (gateway entity dissolves the unknown into the known)
BEAT 3: The Question (the recurring Reddit question — purified demand)
BEAT 4: The Specialist Answer (celestial truth descends into form)
BEAT 5: The Mass Misunderstanding (terrestrial error rises to meet it)
BEAT 6: Objection / Controversy (the polarity that gives the thing life)
BEAT 7: Resolution or Synthesis (the fixed stone — what remains after conflict)
BEAT 8: Return / Hook callback (the ouroboros — end returns to beginning)
```

The dual voice (HOST/AI-COHOST) mirrors the Hermetic dialogue — the master and the student, the celestial and the terrestrial, the fixed and the volatile.

```
HOST (records):      "The scholars call it śaktipāta — a grace that initiates."
AI-COHOST (TTS):     "Wait — so it's not something you achieve? It's something you receive?"
HOST:                "That's exactly the misunderstanding. And it changes everything."
```

Gate: script approved by human (HOST_APPROVAL)

### 6. Distillation (Audio → Visual → Render)

**What it is:** The ascent through the spheres — each pass purifies further.

The script passes through three spheres, each removing more impurity:

```
Sphere 1 — AUDIO:  HOST voice recorded, AI-COHOST via TTS, interleaved
Sphere 2 — VISUAL: Art matched to each beat from the library
Sphere 3 — RENDER: FableCut assembles timeline → video file

What remains after the third sphere: the philosopher's stone — 
a video that communicates celestial truth in terrestrial form.
```

Gate: video rendered and previewed

**What it is:** The stone touches base metal and transforms it. Then the stone itself is transformed by the contact.

The video (stone) touches YouTube (base metal) — and both are transformed:

  Each component updates:
  ┌──────────────────────────────────────────────────────┐

  │     3/3 tests confirmed. Posterior strengthened."     │

  │     Gap score validated for this farm."               │
  │                                                       │

  │    "Beat 4 (specialist answer) has best retention.     │
  │     Lengthen specialist section next video."           │
