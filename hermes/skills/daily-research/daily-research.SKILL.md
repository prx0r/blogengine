# Daily Research

Monitors ~55 YouTube channels in and adjacent to your niche. Every day it checks new uploads, flags breakouts, and stores pattern data. On a rolling basis it surfaces: which topics are trending, which title formats are overperforming, and where the gaps are.

## Why This Exists

You have enough content (56 expansion essays, 50 alchemy shorts, 28 Enneads, 112 VBT meditations) to produce for years. The bottleneck is not content — it's knowing **what to publish next**. This skill tells you:

- "Theurgy is trending right now — publish your theurgy essay"
- "Short form with question titles is outperforming declarative titles 3:1 this month"
- "No one has made a good video on 'barzakh explained' — you have the research, go"
- "Alex O'Connor's video on free will got 2M views — can you do the tantric angle?"

## Analysis Windows

### Daily (55 quota, ~30s runtime)
- Check latest 10 uploads from each of ~55 channels
- Flag any with views_per_day > 2× channel median
- Store in daily JSON
- Run automatically at midnight UTC

### Weekly (300-500 quota, ~2min runtime)
- Pick 5 keywords from RO tags, search YouTube
- Count quality results per keyword
- Flag gap keywords (<2 quality results → film this)
- Compile this week's breakout patterns
- Compare to last 4 weeks: what's changing?

### Monthly (0 quota, analysis only)
- Read last 30 days of daily data
- Top breakout channels (which channels overperform most often)
- Top breakout topics (which topics produce the most breakouts)
- Title pattern shift (are questions overtaking statements?)
- Duration trend (are breakout videos getting shorter/longer?)
- Content direction memo: 3 things you should make next month

### Quarterly (0 quota, strategy)
- Which channels disappeared or went dormant
- Which new channels emerged
- What changed in the niche
- Recommended Pivot (if any)
- Next quarter content roadmap

## Gap Detection Logic

Gap detection answers: "is there demand for this topic, and is anyone serving it?"

**How it works:**
1. Take a keyword from your RO tags (e.g., "barzakh", "pratyabhijna", "spanda")
2. Search YouTube for that keyword + "explained" or "meaning"
3. Count results. If <3 quality videos (not lectures, not 3rd world footage) → **gap confirmed**
4. Bonus: search with `relevanceLanguage=en` to see if English content exists
5. Double bonus: search the term WITHOUT "explained" to measure raw search intent

**Output:**
```json
{
  "keyword": "barzakh explained",
  "total_results": 14,
  "quality_results": 1,
  "best_result": "Ibn Arabi: The Barzakh (Sufi Awakening, 27k views)",
  "gap": true,
  "recommend": "Film a 15-20 min essay — one competitor, and they're small"
}
```

## Breakout Pattern Analysis (the real value)

After 30 days of data, you can answer:

**Title patterns:** "Of 47 breakouts this month, 31 used question titles, 12 used colon-separated titles, 4 used declarative. Question titles had average 3.2× multiplier."

**Topic patterns:** "Consciousness/physics crossovers are 40% of breakouts. Pure philosophy is 25%. Tantra-specific is 15%."

**Channel patterns:** "Small channels (<10k subs) produced 70% of breakouts. The common factor is unique visual style + clear hook."

**Duration patterns:** "10-15 min videos are 3× more likely to break out than 20+ min videos in this niche."

**The meta pattern:** "Breakouts in this niche consistently use 'X is not what you think' or 'The hidden Y' title patterns. They almost never use 'What is X'."

## First Run

For the first run, you need to resolve channel handles to IDs. This is done by the script at startup — it calls `channels.list` with `forHandle` parameter for each handle. This costs ~55 quota once. After that, the IDs are cached in `channels.json`.
