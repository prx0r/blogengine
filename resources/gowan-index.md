## To Extract Full Text

All 122 papers are PDFs. To convert to text for analysis:
```bash
cd /root/projects/blog/gowan-papers
for f in *.pdf; do pdftotext "$f" "$(basename "$f" .pdf).txt" 2>/dev/null; done
```

**Note:** The handbook6 analysis already reviewed these papers. The key findings were that Gowan's Noether reframing is valid, the amplituhedron connection is confirmed, but "light has no gravitational field" and "gravity converts space to time" are falsified/unproven claims.
