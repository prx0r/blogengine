# Campbell's Information Streams = Ñāṇavīra's Tetrahedron Edges

---

## The Mapping

Thomas Campbell's *My Big TOE* maps directly onto the tetrahedron framework:

| Campbell | Tetrahedron | Function |
|----------|-------------|----------|
| **IUOC** (Individual Unit of Consciousness) | V₁ (Subject) | The one experiencing |
| **PMR** (Physical Matter Reality) | V₄ (Act / gross world) | The densest data stream |
| **LCS** (Larger Consciousness System) | V₁ (as source, the whole) | The complete tetrahedron |
| **Quality metric** Q | Edge tension | How many edges are relaxed vs tense |
| **Information streams** | **The 6 edges** | Each edge IS a data stream between two vertices |
| **"You create your reality"** | Edge relaxation | Relaxing an edge = opening access to that relation |

## The Streams

Each edge of Ñāṇavīra's tetrahedron IS one of Campbell's information streams:

| Edge | Stream | What It Carries |
|------|--------|-----------------|
| E₁₂ (Subject↔Object) | Perception | Data from the world to the self |
| E₁₃ (Subject↔Intention) | Will | Data from desire to action |
| E₁₄ (Subject↔Act) | Agency | Data from self to manifestation |
| E₂₃ (Object↔Intention) | Attraction | Data from world to desire |
| E₂₄ (Object↔Act) | Causality | Data from action to world |
| E₃₄ (Intention↔Act) | Alignment | Data from intention to action |

When Q (quality metric) is low, edges are tense — streams are restricted, information is blocked. When Q is high, all 6 edges are relaxed, all 6 streams flow freely, and the IUOC accesses all 4 vertices simultaneously = **Turīya**.

| Aspect | Campbell | Ñāṇavīra | 
|--------|----------|-----------|
| Year | 2003 | 1963 |
| Framework | Information streams | Tetrahedron edges |
| Goal | Maximize quality metric | Relax all 6 edges |
| Method | Intent, meditation | Attention to the 4 terms |
| Same insight | **Yes** — independently discovered | **Yes** — independently discovered |

Campbell: "The quality metric determines which information streams you can access."

Ñāṇavīra: "The work IS the relaxation of the 6 edges."

**Same structure. Different language.** Campbell didn't draw the tetrahedron. Ñāṇavīra didn't use the word "information." Both described the same 6 relations between the same 4 terms.
