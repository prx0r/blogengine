# Analysis: Three Mālas Pack

**File:** `malas_three_veils_pack.zip` (26.8 MB)
**Type:** Animation pack
**Duration:** 38.4s at 10fps
**Scenes:** ~8 scenes at 4.8s each

## What It Teaches

### 1. Filters, Not Substances
The AGENT_KNOWLEDGE_DOSSIER defines a CORE CONSTRAINT:
> "The malas are filters, contractions, and distortions — not demonic substances."

This prevents the most obvious error: depicting spiritual concepts as physical objects. Instead of drawing "evil veils," the visuals show narrowing apertures, refracting prisms, and recursive loops. Each mala has a distinct MECHANISM:

| Mala | Visual Mechanism |
|------|-----------------|
| Āṇava | Narrowing aperture — feels like constriction |
| Māyīya | Refracting prism — feels like splitting |
| Kārma | Recursive feedback wheel — feels like being trapped in a loop |

### 2. Emotional Quality Per Mala
Each visual has a specific FEELING it should evoke, not just a shape:
- Āṇava = narrowing, insufficiency
- Māyīya = refraction, splitting
- Kārma = recursive, action-bound

### 3. Liberation = Transparency, Not Destruction
> "Liberation should render the filters transparent rather than simply destroy the world."

This is a sophisticated constraint. The resolution doesn't smash the veils — it makes them see-through. The world remains, but you see through it.

- Define emotional quality per motif group — not just "what it looks like" but "how it should feel"
- The "transparency not destruction" pattern applies to any liberation/transcendence concept
