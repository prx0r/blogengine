# Analysis: Pramātṛ Taxonomy Pack

**File:** `pramatr_taxonomy_pack.zip` (29.4 MB)
**Type:** Animation pack
**Duration:** ~50s at 10fps
**Scenes:** ~9 scenes at 4.8s each

## What It Teaches

### 1. Progressive Transparency Arc
The seven knowing subjects are ordered by how many FILTERS remain active. As you go up, filters drop away. The visual rule:
> "Show filters becoming progressively transparent upward."

This is a VISUAL NARRATIVE ARC built into the concept itself. Each scene doesn't just show a different subject — it shows the SAME transformation CONTINUING. The viewer watches filters disappear one by one.

### 2. Explicit "Don't"s
> "Avoid a plain seven-rung ladder as the only composition."

This forbids the most obvious, lazy visual choice. My "ladder" system from the Light film was exactly this — a plain vertical axis with rungs. The platinum explicitly REJECTS this.

### 3. Guardrails Prevent Misinterpretation
> "Do not confuse reduced filtering with simple moral ranking."

This prevents a harmful interpretation — that "higher" subjects are "better" people. The visual must not imply hierarchy = morality. I've never included guardrails like this.

### 4. Each Stage Has Specific Qualities
| Stage | Visual Quality |
|-------|---------------|
| Sakala | Caged by overlapping membranes |
| Pralayākala | Voided, not liberated |
| Vijñānākala | Pure yet isolated |
| Mantra-level | Active mantra-form and sovereignty |
| Śiva | Radiant and boundaryless without being visually empty |

Each is a DIFFERENT VISUAL EXPERIENCE. Not just "a brighter version of the previous one."

- Add "guardrails" section that prevents harmful/simplistic interpretations
