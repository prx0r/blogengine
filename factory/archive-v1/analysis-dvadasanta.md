# Analysis: Dvādaśānta Axis Pack

**File:** `dvadasanta_axis_pack.zip` (41.9 MB)
**Type:** Animation pack
**Duration:** 67.2s at 10fps
**Scenes:** ~14 scenes at 4.8s each

## What It Teaches

### 1. Progressive De-localization
The 12 stations move from somatic (heart, throat) → cephalic (eyebrow, crown) → transcendental (above head, external space). This is a transformation arc IN SPACE. The visual changes systematically as it progresses.

### 2. Each Station Has Unique Metaphor
The AGENT_KNOWLEDGE_DOSSIER explicitly forbids repeating generic circles:
> "Each node must have a distinct visual metaphor rather than repeating a generic chakra circle."

Instead: heart lotus, throat tuning fork, palate gate, eyebrow convergence lens, forehead stillness basin, crown aperture. Each is a DIFFERENT VISUAL.

### 3. Style Evolution Chain
```
Prāṇavicāra (previous pack) → horizontal respiratory swing
Dvādaśānta (this pack) → vertical body-to-space migration
```

This documents the visual lineage — what was inherited and what was new.

### 4. ~5s Per Scene, 10fps
The consistent 4.8s scene length with 10fps gives a steady, meditative pace. Each scene is long enough to be understood but short enough to maintain interest.
