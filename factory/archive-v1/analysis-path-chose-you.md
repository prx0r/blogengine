## What It Teaches

### 1. Subtitled + Softsubs Version
This pack includes both `.srt` subtitle file AND a `_softsubs.mp4` version with embedded subtitles. This is a production-quality detail I haven't included.

### 2. Chapter Count Matches Shot Count
109 shots across 10 chapters = ~11 shots per chapter. This is a consistent ratio. My chapters varied wildly (3-12 shots each).

### 3. 41 Motifs For 109 Shots
Each motif appears ~2.7 times on average. This matches the Stones pattern (45 motifs / 106 shots = ~2.4 each). The rule: each motif appears 2-3 times, not 10-15 times.

## Integration Into System

- Generate `.srt` subtitle files alongside final MP4
- Track motifs-per-chapter ratio: no chapter should have more than 15 shots
- Each motif should appear no more than 3 times total
