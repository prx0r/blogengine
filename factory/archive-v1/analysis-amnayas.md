## What It Teaches

### 1. Directional Color Semantics
Each of the four transmission streams has a dedicated color with semantic meaning:
- **East / orange-gold** — emissive, phonemic, creative outpouring
- **South / crimson** — architectural, mantric, protective
- **West / teal** — embodied, vessel-like, transformative
- **North / indigo** — vertical, austere, terminally absorptive
- **Center / ivory-gold bindu** — stable pivot around which all move

This is more specific than my generic palettes. Each color is TIED TO A CONCEPT, not just decoration.

### 2. The Center Holds While Periphery Moves
"the center should remain stable while all four streams move around it" — this is a VISUAL RULE that constrains the composition. I never defined rules like this.

### 3. VISUAL RULES section
The AGENT_KNOWLEDGE_DOSSIER defines concrete rules that prevent generic visuals:
- "Each stream needs a distinct symbolic language" (don't reuse the same motif)
- "East should feel emissive" (specific emotional quality)
- "The center should remain stable" (compositional constraint)
