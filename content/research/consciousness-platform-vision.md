# Consciousness Research Platform — Audio-First Discovery

## The Core Insight

Every paper gets pre-processed into three forms:
1. **JSON** — structured metadata (theory classification, math framework, evidence)
2. **Original PDF** — for academic reference
3. **Audio Narrative** — pre-generated engaging story version (big model, offline)

This means users never read a paper raw. They experience it as a podcast-quality narrative with the JSON underneath for reference.

## The Triple Pipeline

```
Paper Upload
  ↓
JSON Classification (consciousness theory metadata)
  ↓
Narrative Generation (70B model, offline, no rush)
  ↓
Professional TTS
  ↓
Modular Storage (hook, context, theory, methodology, results, implications, connections)
  ↓
Available for journeys
```

## Narrative Structure (Standardized)

Every paper gets the same modular structure for consistent quality:

1. **Hook** (30s) — "What if consciousness is the universe computing itself?"
2. **Context** (2m) — "This paper emerged from a heated debate between..."
3. **Core Theory** (5m) — "The researchers proposed something radical..."
4. **Methodology** (3m) — "When they tested this with brain scans..."
5. **Implications** (3m) — "This changes everything about..."
6. **Connections** (2m) — "This links to three other theories..."
