# The Path of Re-Rendering

## Master definition

Ordinary consciousness is a conditioned rendering regime.

It is not raw reality. It is a stabilized interface produced by body, nervous system,
perception, language, culture, class, media, memory, craving, fear, and self-reference.

Practice is the progressive deconstruction and re-ordering of that rendering.

Progress means: less compulsion, less reification, clearer perception, more embodied awareness,
richer imaginal capacity, cleaner symbolic ordering, deeper ethical responsiveness,
and more concrete incarnation in life.

## The 17 Phases

### Phase 0 — The premodern metaphysical baseline: world as order
Before modernity, most serious systems assumed cosmos: an ordered whole in which mind, form,
number, word, soul, nature, and divinity are linked.

Key thinkers: Newton, Leibniz, Kant, German Idealism, Marx, Tesla, von Neumann, Turing.

### Phase 1 — Break naive realism: the world is a dashboard
Perception is not a transparent window. It is an active interface.
Key sources: Kant, Hoffman, Seth, Clark, Friston, Baudrillard, Debord, Marx, Levin.

### Phase 2 — Break crude materialism: matter is structure, not dead stuff
Modern physics dissolves crude materialism. Reality appears as fields, quantum states,
geometry, symmetry, information, emergence.
Key sources: Newton, Einstein, Wallace, Deutsch, Wolfram, Connes, Arkani-Hamed, Bohm.

### Phase 3 — Break mind-as-computer: computation is not enough
Key sources: Turing, von Neumann, Dennett, Frankish, Metzinger, Chalmers, Penrose.

### Phase 4 — Expand agency: life is nested cognition
Key sources: Levin, Friston, Varela, Thompson, Kauffman, Hoffman, Leibniz, Whitehead.

### Phase 5 — Mechanicality: the ego-machine runs automatically
Key sources: Gurdjieff, Ouspensky, Krishnamurti, Bohm, Metzinger, Dennett.

### Phase 6 — Dependent arising: the operational exit
The central phase. Contact → feeling-tone → sign → craving → papañca → selfing → view.
Key sources: Ñāṇananda, Ñāṇavīra, Burbea.

### Phase 7 — Madhyamaka: no essence anywhere
Key sources: Nāgārjuna, Candrakīrti, Śāntideva, Tsongkhapa, Mipham.

### Phase 8 — Dzogchen/Mahāmudrā: do not construct liberation
Key sources: Longchenpa, Patrul Rinpoche, Dudjom Lingpa, Tulku Urgyen, Namkhai Norbu.

### Phase 9 — Pramāṇa and language: valid knowing, naming, mantra
Key sources: Nyāya, Dignāga, Dharmakīrti, Bhartṛhari, Abhinavagupta, Tsongkhapa.

### Phase 10 — Imaginal reconstruction: image is empty but potent
**THE STARTING POINT FOR ACQUISITION.**
Key sources: Jung, Corbin, Hillman, Burbea, rasa theory, Assagioli, Harpur.

### Phase 11 — Body-energy: the body is the living interface
Key sources: Patañjali, Haṭha Yoga, Vijñāna Bhairava, Abhinavagupta, Wallis, Vajrayāna.

### Phase 12 — Ritual and ceremonial technology
Key sources: Iamblichus, Proclus, Chaldean Oracles, Ficino, Picatrix, Agrippa, Bardon.

### Phase 13 — Daimon, HGA, Perfect Nature, guidance
Key sources: Plato, Iamblichus, Greenbaum, Picatrix, Abramelin, Regardie, Jung, Harpur.

### Phase 14 — Nonordinary rendering: dreams, astral
Key sources: Monroe, Campbell, Tart, Thompson, Tibetan dream yoga.

### Phase 15 — Shamanism and ecological imagination
Key sources: Eliade, Viveiros de Castro, Abram, Harvey.

### Phase 16 — Visionary cosmologies: Steiner, Sheldrake, Faggin
Key sources: Steiner, Roberts/Seth, Sheldrake, Faggin.

### Phase 17 — Social and historical incarnation
Key sources: Hegel, Marx, Debord, Baudrillard, Taylor, Barfield, Gebser.

---

## Acquisition Priority

Start from Phase 10 (Imaginal) and expand outward in natural connection order:

Phase 10 (Imaginal) → Phase 12 (Ritual) → Phase 13 (Daimon) → Phase 11 (Body-energy) → Phase 9 (Language/Mantra) → Phase 6 (Dependent arising)

Do NOT jump to Phase 2 (physics) or Phase 16 (visionary cosmologies) until the core imaginal-ritual-daimonic corpus is well-covered.
