2. State Representation: The Listener's Beliefs
The framework models the listener's cognitive state as a set of probability distributions.

2.1. The Learned Schema: Transition Probability Matrix (TPM)
The system is loaded with a pre-computed TPM that represents long-term, culturally learned musical knowledge. This matrix defines the probability of transitioning from one musical state to another.

State C_i: A state is defined by a tuple of functional harmony tokens, representing the recent context. For a 2nd-order model: C_i = (token_{t-1}, token_{t}). (e.g., (ii, V7)).

TPM: A key-value store where TPM(C_i) returns a probability vector over all possible next tokens C_j.

2.2. The Prior Belief P(t)
This is the model's expectation for the next time step, t, generated before observing the audio at t. It is a probability distribution over all possible future states, retrieved from the TPM based on the state at t-1.

This is the model's perception of the event that actually occurs at time t. It is generated by the real-time Music Information Retrieval (MIR) front-end, which outputs a probability distribution over possible states given the signal S(t).

5. Working Example: A ii-V-I Cadence in C Major
Scenario: The music is in the key of C Major. We have just heard a Dm chord (ii) and now, at t-1, a G7 chord (V7) begins to play. We want to calculate the emotional state at time t when the music resolves to Cmaj (I).

Context: The state at t-1 is C(t-1) = (ii, V7).

Calculate Prior P(t): We query the TPM with this state. The model has learned that V7 in this context resolves to I most of the time. Let's assume the TPM returns:

P(I | (ii,V7)) = 0.85

P(vi | (ii,V7)) = 0.10

P(other | (ii,V7)) = 0.05
This distribution is our prior belief P(t).

Calculate Tension T(t-1): We calculate the entropy of P(t).

H(P(t)) = -(0.85\log_2(0.85) + 0.1\log_2(0.1) + ... ) \approx 0.78

Assuming a functional vocabulary of N=40, log_2(40) \approx 5.32.

Tension(t-1) = 0.78 / 5.32 = 0.147. There is low but non-zero tension; a resolution is strongly expected.

The Event at t: A Cmaj chord is played.

Sensory Evidence Q(t): The MIR module hears the Cmaj chord with high confidence.

Q(I) = 0.98

Q(other) = 0.02

Calculate Surprise S(t): We calculate the KL Divergence D_KL(Q(t) || P(t)).

Surprise(t) = 0.98 * log₂(0.98 / 0.85) + ... \approx 0.20.

This is a very low surprise value, as the event was highly probable.

Calculate New Tension T(t): The state is now C(t) = (V7, I). Let's assume the TPM predicts many possibilities after a I chord, leading to a higher entropy H(P(t+1)) \approx 2.5.

Tension(t) = 2.5 / 5.32 = 0.47. Tension has risen for the next event, but dropped for the current one. The key for valence is the drop from the previous state.

Calculate Valence V(t): We assume w_s=0.2, w_c=0.5, w_p=0.3 and k_p=2.0.

V_s(t): Cmaj is highly consonant. V_s(t) = 0.9.

V_c(t): It's the I chord (Tonic), the most stable function. V_c(t) = 1.0.

V_p(t): Based on the change in tension into this moment. The transition from the tension of V7 to the stability of I represents a drop in predictive tension for this moment. We can approximate the previous tension Tension(t-1) as 0.147 and the current moment's resolved tension as near zero.
V_p(t) = tanh(2.0 * (0.147 - 0)) = tanh(0.294) = 0.286.
(Note: A more rigorous model would have a continuous tension value, but this step-change illustrates the principle.)

Final V(t): (0.2 * 0.9) + (0.5 * 1.0) + (0.3 * 0.286)
= 0.18 + 0.5 + 0.0858
= 0.7658

Conclusion of Example: The framework outputs a high positive valence (+0.766), accurately modeling the strong feeling of pleasure and resolution from a perfectly executed ii-V-I cadence. The low surprise and positive predictive valence (V_p) show that this pleasure comes not from novelty, but from the satisfying confirmation of a strongly held expectation.
