# Kimi Vision Analysis — Seed Image Review

Please analyze these seed images for our AI video generation pipeline. We use LTX-2.3 to generate 10-15 second video clips in a **stop-motion/clay-texture aesthetic** (think Guillermo del Toro, Coraline, Fantastic Mr. Fox).

## Our Visual Style

```
Aesthetic: Stop-motion, clay-textured, handcrafted, warm parchment tones
Palette:  Muted, warm, slight grain, geometric simplification
Lighting: Volumetric, single source, dramatic shadows
Subjects: Architecture, landscapes, abstract concepts, geometric diagrams
```

## What We Need From You

For each world below, review the image URLs and tell us:

1. **Which 3-5 images best represent the world's atmosphere?** (lighting, color palette, architecture style)
2. **Which images would make good LTX-to-video backgrounds?** (clear structure, identifiable forms, dramatic lighting)
3. **Which images are too detailed/complex for LTX?** (intricate patterns the model can't reproduce)
4. **What keywords should we add to LTX prompts** to reproduce the mood/texture of each image?
