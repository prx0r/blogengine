# Concept Explainers — How LTX Teaches Each Idea

## The Two Modes

| Mode | Input | Best for | Reliability |
|------|-------|----------|-------------|
| **Image-to-Video** | Seed image + prompt describing motion | World atmospherics, deity visuals, temple scenes | HIGH — the image anchors the output |
| **Text-to-Video** | Prompt only | Abstract concepts (spanda, rasa, tattvas) | MEDIUM — needs precise prompt engineering |

**Rule:** Always use Image-to-Video when we have a seed image. Only use Text-to-Video for pure abstractions.

## Standardized Explainer Format

Each concept video in the series follows:

```
1. WORLD ESTABLISH (15s I2V)       → Sets the tradition's atmosphere
2. THE QUESTION (10s D or TEXT)    → What the Seeker wants to know
3. THE TEACHING (15-20s I2V/T2V)  → The core visual explanation
4. THE IMPLICATION (10s E)        → The Seeker witnesses/shows change
5. THE RETURN (10s I2V)           → Back to the world, but something shifted
```

Total per concept: ~60-75s = **roughly 20 concepts fill a 20-min video.**
